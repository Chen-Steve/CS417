# About The Game
This is a VR game created on Unity 6.3 and tested on the Meta Quest 3.<br/>
A cozy restaurant sim run by friendly goblins, now filled with new juicy feedback! Manage your restaurant by cooking delicious food, upgrading your kitchen, and serving happy customers!<br/>
